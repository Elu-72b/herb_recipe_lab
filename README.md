# **ハーブブレンド管理アプリ** Herb Recipe Lab

# **1. サービス概要**

このサービスは、ハーブティーのブレンドレシピを記録・整理・共有できるアプリです。
ハーブの配合、味のメモ、期待する効果などをまとめて管理できます。
ハーブブレンドを楽しむ人が、自分のレシピを再現可能な形で記録し、経験を蓄積できることを目的としたサービスです。

# **2. このアイデアはどこから生まれたか**

# **2-1. きっかけとなった体験・感情**

ハーブティーのブレンドを自分で作るようになってから、「美味しかったブレンドの配合を忘れてしまう」、「好みではないブレンドを覚えてたつもりが忘れてしまう」ことがありました。
スマートフォンのメモや紙のノートに記録していましたが、

- どのハーブをどの割合で入れたか
- 味や香りの印象
- 体調への影響
  などを体系的に整理するのが難しく、同じブレンドを再現できないことに不便さを感じました。
  また、他の人が作ったハーブブレンドを参考にしたいと思っても、体系的に整理されているサービスが少ないことにも気づきました。

# **2-2. なぜそれが「気になった」のか**

ハーブティーは、味や香りだけでなく、体調や気分に合わせてブレンドを調整する楽しさがあります。
しかし、ブレンド内容を適切に記録していないと、

- 過去の経験を活かせない
- 同じブレンドを再現できない
- 試行錯誤の結果が蓄積されない
  という問題があります。
  ハーブブレンドは経験の積み重ねが重要であるにも関わらず、その経験を体系的に記録できるツールが少ないことに違和感を感じました。

# **3. 課題の整理**

# **3-1. 表に見えている困りごと**

ハーブティーのブレンド内容を整理して記録できる専用ツールが少ない。
また、ネット検索しても目当てのブレンドレシピがなかなかヒットしない。

# **3-2. 本当に解決したい課題**

- ブレンド内容を忘れてしまう。
- 味・香り・配合・効果などを一緒に記録できる仕組みがない。
- ハーブブレンドの経験を蓄積し、再現できる形で保存するツールが少ない。

→ 本当に解決したい課題
ハーブブレンドの経験や知識を、再現可能な形で記録・蓄積・共有できないこと。

# **4. 想定ユーザーについて**

# **4-1 想定しているユーザー**

①ハーブティーが好きなブレンド初心者
②日常的にハーブをブレンドするユーザー

**生活・状況設定**

- 生活・状況：自宅でハーブティーを飲む習慣がある、ハーブのブレンドを試すことが好き
- 理由：ブレンドを試行錯誤する中で、配合・味の印象・効果 などを記録したくなる場面が多いため。

**環境設定**：

- 環境：自宅でハーブティーを楽しんでいる人
- 理由：自宅では複数のハーブを使ってブレンドする機会が多く、配合管理の必要性が高いため。

# **4-2 自分とユーザーの距離**

自分自身がハーブブレンドを試す中で、レシピや味の記録方法や、手元にあるハーブの種類に合わせたブレンドをレシピメモから抽出することに不便さを感じた経験があったため。

# **4-3 実際に使われる可能性について**

最初に使ってくれそうな人：ハーブティーのブレンドを趣味としている人。
どんな場面で思い出すか：新しいハーブブレンドを作ったとき。

今使っている代替手段：

- スマートフォンのメモ
- 紙のノート
- Chat GPT

それを置き換えてまで使う理由：
ハーブブレンドの

- 配合
- 味メモ
- 効果タグ
- ブレンドを作ってみた感想
  などを1つの場所で整理できるため。

# **4-4 ユーザーが導入して利用するまで**

**サービスを知るきっかけ**

- ハーブティーのブレンドレシピを検索するとき。

**最初の一歩**

- ブレンドレシピを検索

**利用を続ける理由**

- 他の人のブレンドを参考にできる
- 自分の好きなブレンド履歴が蓄積される
- 手持ちのハーブを使ったブレンドを検索できる

# **5. 既存サービス・競合調査**

# **5-1 似たサービス**

①Cookpad
URL：[https://cookpad.com/jp](https://cookpad.com/jp)
できること：レシピ投稿、レシピ検索、レシピ共有

②母乳育児支援ハーブ専門店 ハーブガーデンショップ
URL：[https://www.e081.com/herb-diagnosis.html](https://www.e081.com/herb-diagnosis.html)

# **5-2 それでも自分が作りたい理由**

既存のレシピサービスは主に料理レシピを対象としており、

- ハーブブレンドの配合
- 味の印象
- 体調への効果
  などを体系的に整理することには向いていないと感じました。
  ハーブブレンドに特化した記録と共有の仕組みを持つサービスが少ないことが自分で作りたいと思った理由です。

# **5-3 差別化**

このアプリは、
ハーブティーをブレンドする人が
自分のレシピと味の記録を体系的に蓄積し、
他ユーザーと共有することができます。

# **6. このサービスで提供したい価値**

# **6-1 ユーザーの変化**

- 行動の変化：ブレンドレシピを継続的に記録するようになる。
- 気持ちの変化：自分のブレンド経験が知識として蓄積される安心感が生まれる。
- 考え方の変化：ハーブブレンドを試行錯誤を楽しむ「実験」のように考えられるようになる。

# **6-2 価値を一文で表す**

このサービスは、
ハーブブレンドをする人が、自分や他人のレシピを記録・再現できるようになるサービスです。

# **8. このアプリの懸念点とその対策**

# **懸念点①**

- 何が問題になりそうか：ハーブブレンドをするユーザー数が限定的である可能性。

- なぜそう思うか：ハーブブレンドは一般的な料理や飲み物と比べ、趣味人口が比較的少ない分野であるため。

- 対策：ハーブブレンドだけでなく、市販ハーブティーの飲用記録を投稿できるようにすることでハーブティー初心者でも飲んでみたいと思ってもらえるようにする。

- 香りのチャート表示
