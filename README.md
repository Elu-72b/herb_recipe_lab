# ハーブブレンド管理アプリ Herb Recipe Lab

# **1. サービス概要**

このサービスは、ハーブティーのブレンドレシピを記録・整理・共有できるアプリです。
ハーブの配合、味のメモ、期待する効果などをまとめて管理できます。
ハーブブレンドを楽しむ人が、自分のレシピを再現可能な形で記録し、経験を蓄積できることを目的としたサービスです。

# 2. このアイデアはどこから生まれたか

## 2-1. きっかけとなった体験・感情

ハーブティーのブレンドを自分で作るようになってから、「美味しかったブレンドの配合を忘れてしまう」、「好みではないブレンドを覚えてたつもりが忘れてしまう」ことがありました。
スマートフォンのメモや紙のノートに記録していましたが、

- どのハーブをどの割合で入れたか
- 味や香りの印象
- 体調への影響
  などを体系的に整理するのが難しく、同じブレンドを再現できないことに不便さを感じました。
  また、他の人が作ったハーブブレンドを参考にしたいと思っても、体系的に整理されているサービスが少ないことにも気づきました。

## 2-2. なぜそれが「気になった」のか

ハーブティーは、味や香りだけでなく、体調や気分に合わせてブレンドを調整する楽しさがあります。
しかし、ブレンド内容を適切に記録していないと、

- 過去の経験を活かせない
- 同じブレンドを再現できない
- 試行錯誤の結果が蓄積されない
  という問題があります。
  ハーブブレンドは経験の積み重ねが重要であるにも関わらず、その経験を体系的に記録できるツールが少ないことに違和感を感じました。

# 3. 課題の整理

## 3-1. 表に見えている困りごと

ハーブティーのブレンド内容を整理して記録できる専用ツールが少ない。
また、ネット検索しても目当てのブレンドレシピがなかなかヒットしない。

## 3-2. 本当に解決したい課題

- ブレンド内容を忘れてしまう。
- 味・香り・配合・効果などを一緒に記録できる仕組みがない。
- ハーブブレンドの経験を蓄積し、再現できる形で保存するツールが少ない。

→ 本当に解決したい課題
ハーブブレンドの経験や知識を、再現可能な形で記録・蓄積・共有できないこと。

## 5-2 それでも自分が作りたい理由

既存のレシピサービスは主に料理レシピを対象としており、

- ハーブブレンドの配合
- 味の印象
- 体調への効果
  などを体系的に整理することには向いていないと感じました。
  ハーブブレンドに特化した記録と共有の仕組みを持つサービスが少ないことが自分で作りたいと思った理由です。

## 5-3 差別化

本アプリは、単なるレシピ記録を超え、**「個人の嗜好に徹底的に寄り添う」**体験を提供します。

ポジティブな味ベース検索：
効能だけでなく「甘み」「スパイシー」「フローラル」といった味の好みからレシピに辿り着ける設計。

ブレンド検索時の除外フィルタリング：
苦手なハーブや禁忌（カフェイン、特定成分）をあらかじめ設定することで、ブレンドを検索時に「このブレンドは苦手なハーブが入ってるから飲めないな」と諦めることがなくなります。

嗜好の資産化：
「この配合は好き」「これはミントが強すぎた」という試行錯誤を蓄積し、自分だけの「至高の一杯」を実現させ、またそれを再現可能にします。

「購買」ではなく「研究（記録）」への特化:
既存のサイトの多くは「売るためのレシピ」や「完成された商品」が主役です。対して「Herb Recipe Lab」は、「手元の素材でどう美味しくするか」という試行錯誤の過程そのものを記録することができます。

# 7. このアプリで実現すること

## MVPで作る機能

- ユーザー認証機能
  メールアドレスによる新規登録、ログイン、ログアウト機能。
- ハーブ管理機能（マスタ・自由入力）
  50音順に整理されたハーブマスタから選択。
- マスタにない材料は「その他」として自由記述で追加可能。
- 二段階保存によるブレンド研究記録
  STEP 1（淹れる前）: 日付、タイトル、ハーブ配合、200mlあたりの分量（g/小さじ単位統一切り替え）を記録。
  STEP 2（飲用後）: 実際の味や香りの感想を、レシピを振り返りながら追記。
- レシピ共有機能
  投稿されたレシピの一覧表示、詳細表示。

## 本リリースで作る機能

- 五段階評価 ＆ 7要素の風味チャート
  「甘味・酸味・清涼感・華やかさ・草土・スパイシー・コク苦味」の7指標で味を客観的に可視化。
  自己分析や「好みの近いユーザー」とのマッチングの基礎データとなります。
- ハーブ辞典（Turbo Stream実装）
  レシピ入力画面を離れることなく、非同期でハーブの画像・風味・効能・禁忌を確認できる機能。
  50音順、風味ジャンル、効能ジャンルでの並び替え・絞り込みに対応。
- 多角的なレシピ検索機能
  使用ハーブ、期待する効果、好みの風味タグを組み合わせたクロス検索。
- コミュニティ・ストック機能
  ユーザー間のフォロー機能、ブックマーク、他ユーザーのブレンドレシピの評価機能。
- 外部API（Gemini API）連携によるブレンド提案
  手持ちのハーブから、相性の良い組み合わせや期待できる効果をAIがアドバイス。
- 市販ハーブティー・飲用ログ機能
  自作レシピだけでなく、既製品の飲用感も同様のチャートで記録可能に。

# 8. このアプリの懸念点とその対策

## 懸念点

- 何が問題になりそうか：ハーブブレンドをするユーザー数が限定的である可能性。

- なぜそう思うか：ハーブブレンドは一般的な料理や飲み物と比べ、趣味人口が比較的少ない分野であるため。

- 対策：ハーブブレンドだけでなく、市販ハーブティーの飲用記録を投稿できるようにすることでハーブティー初心者でも飲んでみたいと思ってもらえるようにする。
