## Issue 23: ハーブ図鑑の新規作成・編集導線の制限（管理者権限導入の準備）

- **概要**: ハーブマスタデータの正確性を維持するため、一般ユーザーの画面からハーブ図鑑の「新規作成」および「編集・削除」ボタンを非表示にします。これは将来的な管理者権限（Admin）導入に向けた第1段階の対応です。
- **背景**:
  - 現在、ログインユーザーであれば誰でもハーブ図鑑の追加や修正ができてしまう。
  - 誤った情報の登録や、意図しないデータの書き換えを防ぐ必要がある。
  - ハーブ図鑑の編集権限は作成者のみに付与されるため、詳細画面のボタンは変更しない。
- **タスク**:
  - [x] **ハーブ図鑑一覧（index）の修正**: 「新規ハーブ登録」ボタンをコメントアウト。
  - [x] **管理者権限導入の準備**: 将来的に `if current_user.admin?` で制御するための構造を意識したコード配置にする。
- **完了条件**:
  - [x] ログイン後、ハーブ一覧画面に「新規作成」ボタンが表示されないこと。
